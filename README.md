# Como executar este projeto

```bash
# clone o repositório
git clone https://github.com/isaacbrds/todo_list.git 

# acesse a pasta
cd todo_list

# Instale as gems
bundle install

# Crie o banco de dados
rails db:create db:prepare db:migrate

```
